知识点：
1.streamlit要用虚拟环境启动，否则完全无法启动

2.配置虚拟环境

         virtualenv venv //在当前目录生成虚拟环境
         cd D:\pythonProject1\venv\Scripts //进入虚拟环境的启动文件夹
        D:\pythonProject1\venv\Scripts\active //启动虚拟环境
        deactivate //退出虚拟环境

3.官方文档

        Get started - Streamlit Docs

        国内教程 Streamlit开发手册 / 导入Streamlit - 汇智网 (hubwiz.com)

问题：
1.streamlit run 文件时经常会出现cmd识别不到streamlit这个命令的报错，设置系统环境变量，以管理员身份运行都不行

        要以虚拟环境运行的方式运行
